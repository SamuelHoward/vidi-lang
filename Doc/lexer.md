This document gives information relevant to the vidi lexer

# Complex Tokens:

Boolean Values:
Tok_Bool of bool

Integer Values:
Tok_Int of int

Identifiers:
Tok_ID of string

String Values:
Tok_Str of string

Float Values:
Tok_Float of float
